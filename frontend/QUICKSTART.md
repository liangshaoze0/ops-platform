# 前端快速启动指南

## 1. 安装依赖

```bash
cd frontend
npm install
```

## 2. 启动开发服务器

```bash
npm run dev
```

前端将在 `http://localhost:3000` 启动。

## 3. 确保后端服务运行

前端需要后端API支持，确保后端服务在 `http://localhost:8080` 运行。

## 4. 访问应用

打开浏览器访问：`http://localhost:3000`

## 功能测试

### 登录流程

1. 访问首页，自动跳转到登录页
2. 点击"使用 GitHub 登录"按钮
3. 授权后自动跳转回应用并显示主面板

### 生成背景图

1. 在主面板的"AI背景图生成"卡片中
2. 选择或输入尺寸
3. （可选）输入描述提示词
4. 点击"生成背景图"
5. 等待生成完成，背景图会自动应用为页面背景

### 查看用户信息

主面板右侧显示当前登录用户的详细信息。

## 开发说明

### 热重载

Vite支持热模块替换（HMR），修改代码后自动刷新。

### API代理

开发环境下，所有 `/api` 请求会自动代理到 `http://localhost:8080`。

### 环境变量

如需修改后端地址，编辑 `vite.config.js`：

```javascript
proxy: {
  '/api': {
    target: 'http://your-backend-url:8080',
    changeOrigin: true,
  }
}
```

## 常见问题

### 1. 登录后无法获取用户信息

- 检查后端服务是否运行
- 检查浏览器控制台是否有错误
- 确认token已正确存储

### 2. 背景图生成失败

- 检查网络连接
- 确认后端API正常
- 查看浏览器控制台错误信息

### 3. 样式显示异常

- 清除浏览器缓存
- 重启开发服务器
- 检查CSS文件是否正确加载
